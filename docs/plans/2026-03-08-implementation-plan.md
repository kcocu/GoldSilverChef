# GoldSilver 요리사 — 구현 계획

## Phase 1: 프로젝트 셋업 & 핵심 데이터 모델
1. Flutter 프로젝트 생성 (웹 + Android + iOS)
2. 폴더 구조 설정
   ```
   lib/
     models/        # 데이터 모델
     services/      # 게임 로직
     screens/       # 화면
     widgets/       # 공통 위젯
     data/          # 레시피 JSON
   ```
3. 핵심 모델 정의
   - `Ingredient` (재료: id, name, type, icon)
   - `Recipe` (레시피: id, inputs[], tool_ranges{}, output, quality_formula)
   - `CookingResult` (결과: recipe, quality_grade, scores{})
   - `Judge` (심사위원: name, weight_map{})
   - `NpcChef` (NPC: name, skill_level, recipes_known[])

## Phase 2: 조합 엔진
1. `CraftingEngine` — 재료 조합 매칭
   - 입력 재료 목록 → 레시피 DB 검색 → 결과물 반환
   - 연쇄 조합 지원 (결과물이 다시 재료로 사용)
2. `QualityCalculator` — 품질 계산
   - 불/칼/물 수치 vs 레시피 최적 범위 → F~SS+ 등급
   - 중간 품질 반영 공식
3. 레시피 JSON 구조 설계 & 샘플 데이터 50종 작성

## Phase 3: 조리 UI & 인터랙션
1. **조리대 화면** — 재료 배치 영역 + 도구 영역
2. **칼 조작** — 터치/클릭 카운터
3. **물 조작** — 드래그 기울기 감지 (게이지 미표시)
4. **불 조작** — 1/2/3 단계 선택 + 프라이팬 위 타이머
5. **조합 애니메이션** — 재료 합치기 효과
6. **결과 표시** — 완성된 요리 + 품질 등급

## Phase 4: 심사 시스템
1. `JudgingEngine` — 채점 로직
   - 돌고래: 맛(30%) + 완성도(40%) + 등급(30%)
   - 흑설탕: 맛(20%) + 창의성(30%) + 비주얼(20%) + 등급(30%)
2. 점수 산출 공식 구현
   - 맛: 레시피 난이도 x 품질
   - 완성도: 도구 수치 정확도
   - 창의성: 조합 단계 수 + 희귀 재료 사용
   - 비주얼: 품질 등급 + 재료 다양성
3. 심사 결과 화면 (돌고래/흑설탕 점수 + 코멘트)

## Phase 5: 자유 경연 모드
1. 자유 요리 화면 (시간 설정 옵션)
2. 재료 선택 → 조리 → 심사 플로우
3. 레시피북 (자동 기록 + 즐겨찾기)

## Phase 6: 스토리 모드
1. `StoryModeManager` — 8단계 진행 관리
2. NPC 시스템
   - 랜덤 실력 + 단계별 난이도 스케일링
   - NPC 요리 결과 자동 생성
3. 주제 시스템
   - 랜덤 주제 선정 + 조합 힌트 표시
4. 각 단계 구현
   - 1단계: 점수 컷 (100→40)
   - 2단계: 1:1 (40→20)
   - 3단계: 팀전 (20→14)
   - 4단계: 개인 + 분기 (14→10)
   - 5단계: 1:1 (10→5)
   - 6단계: 자유 + 분기 (7명)
   - 7단계: 요리 지옥 x5판 (6→1)
   - 8단계: 결승, 만장일치 (2명)
5. 스테이지 간 결과/진행 화면

## Phase 7: 랜덤 모드
1. 레시피 조합 셔플 로직
2. 랜덤 모드 전용 UI

## Phase 8: 리더보드 & 저장
1. 로컬 저장 (Hive/SharedPreferences)
   - 레시피북, 스토리 진행, 설정
2. 온라인 리더보드 (Firebase or 간단 백엔드)
   - 자유 경연 점수 랭킹

## Phase 9: 폴리싱
1. 사운드/효과음
2. 요리 아이콘/이미지
3. 튜토리얼 (첫 플레이 가이드)
4. 설정 화면 (언어: 한국어 기본, 영어 추후)

## 우선순위
- **MVP (Phase 1~5)**: 조합 + 조리 + 심사 + 자유 경연
- **Core (Phase 6~7)**: 스토리 + 랜덤 모드
- **Launch (Phase 8~9)**: 리더보드 + 폴리싱
