# GoldSilver 요리사 — 설계 문서

## 개요
Little Alchemy 2 스타일의 요리 조합 + 경연 게임.
24종 기본 재료를 조합하여 10,000종 요리를 만들고, 심사위원에게 채점받는 싱글플레이 게임.

## 플랫폼
- **Flutter** (웹 + 모바일 동시)

## 언어
- **한국어** 기본
- 영어는 추후 설정에서 지원 예정

## 기본 재료 (24종)

### 현실 계열 (13종)
곡물, 육류, 수산물, 알, 유제품, 채소, 과일, 버섯, 기름, 액체, 설탕, 소금, 양념

### 추상 계열 (11종)
빨간가루, 파란가루, 초록가루, 보라가루, 노란가루, 모래시계, 신선한 구슬, 활발한 구슬, 날아라 구슬, 흘러라 구슬, 정체모를 가루

### 조합 원리
- 색상 가루: 기본 재료와 합쳐 구체적 재료로 변환 (육류+빨간가루→소고기)
- 구슬: 합쳐서 다른 종류로 변환, 구슬+불→꿀/수액 등
- 모래시계: 숙성 재료 (장, 빙하수, 금 등 시간이 필요한 재료)
- 정체모를 가루: 특수/이색 재료 파생
- 연쇄 조합: 중간 결과물이 다시 재료가 됨

### 조합 예시
```
육류 + 빨간가루 → 소고기
곡물 + 초록가루 + 신선한 구슬 → 콩
물 + 물 + 소금 → 바닷물
콩 + 바닷물 → 두부
구슬 + 불 → 꿀/수액
양념 + 알 → 마요네즈
재료 + 모래시계 → 장/빙하수/금(숙성류)
```

## 조리 도구 & 조작

| 도구 | 조작 방식 | 수치 결정 |
|------|-----------|-----------|
| 칼 | 터치/클릭 횟수 | 횟수 |
| 물 | 드래그로 기울여 붓기 | 기울기 정도 (게이지 미표시) |
| 불 | 단계 선택 (1/2/3) | 단계 x 시간 |
| 프라이팬/냄비 등 | 재료를 올려놓는 용기 | 불과 조합하여 시간 측정 |

## 품질 시스템
- 불/칼/물 수치의 정확도에 따라 **F ~ SS+** 등급
- 중간 단계의 품질이 최종 결과에 영향
- 레시피별 각 도구의 최적 수치 범위가 정의됨

## 심사위원 (고정 2명)

| 이름 | 중시 항목 |
|------|-----------|
| 돌고래 | 맛 + **완성도** + 등급 |
| 흑설탕 | 맛 + **창의성** + **비주얼** + 등급 |

## 게임 모드

### 스토리 모드 (100명 → 1명, 8단계)

| 단계 | 형식 | 인원 변화 | 진행 |
|------|------|-----------|------|
| 1 | 주제 랜덤, 개인 | 100→40 | 일정 점수 이상 합격 |
| 2 | 주제 랜덤, 1:1 | 40→20 | 대결 승리 |
| 3 | 주제 랜덤, 팀전(사실상 1:1) | 20→14 | 상위 4명 안에 들면 통과 |
| 4 | 주제 랜덤, 개인 | 14→10 | 1등→6단계 직행, 꼴지→탈락, 나머지→5단계 |
| 5 | 주제 랜덤, 1:1 | 10→5 | 대결 승리 |
| 6 | 주제 없음(자유), 개인 | 7명 | 1등→8단계 직행, 나머지 6명→7단계 |
| 7 | 요리 지옥, 주제 하나 고정 | 6→1 | 꼴지 탈락 x5판, 최종 생존자→8단계 |
| 8 | 주제 없음, 결승 | 2명 | 만장일치 필요, 아니면 재대결(같은 주재료 사용 금지) |

- NPC: 랜덤 실력 + 단계가 올라갈수록 더 좋은 요리
- 주제 재료가 정해지면 해당 재료의 조합 경로 힌트 제공
- 추후 난이도 조절 기능 추가 예정

### 자유 경연
- 시간 설정 자유 (또는 무제한)
- 자유롭게 요리 후 심사위원 채점

### 랜덤 모드
- 요리 조합이 완전 랜덤으로 변경됨
